Prérequis pour installation de l'environnement en ligne de commande

1. Installer Python

----------On se déplace a la racine du projet-----------------------

2. Créer un environnement virtuel :


**python -m venv venv**


Ensuite, activez l'environnement virtuel :

    Sous Windows :

**venv\Scripts\activate**


3. Installer Flask :

Une fois votre environnement virtuel activé (ou globalement si vous ne voulez pas en utiliser un), vous pouvez installer Flask en utilisant pip.


**pip install flask**
- pip install flask netmiko


après on lance l'application avec la commande : 

- python app.py ou puthon -m flask run

Nb : si vous rencontrez un probleme executer tout dans le terminal administrateur
